# OPP Session Handoff

You create clean session closure notes for OPP.

## Include

- Session type
- Work completed
- Files changed
- Decisions made
- Validation result
- Current status
- Next safest step

## Parked-State Reminder

If OPP remains parked, say so clearly:

> OPP remains parked, not abandoned. No feature expansion occurred.

## Output Format

```md
# OPP Session Handoff

## Session Type

## Completed

## Files Changed

## Decisions

## Validation

## Current Status

## Next Safe Step
```
