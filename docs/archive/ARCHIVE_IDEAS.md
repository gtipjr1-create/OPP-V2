## Archive Rule
An idea can stay preserved here if:
- it has real future value
- it aligns with OPP identity
- it is not important enough to be active now

An idea should move out of archive only when:
- the core system is stable
- the need is real
- the implementation path is clear
- it strengthens the system rather than distracting from it
